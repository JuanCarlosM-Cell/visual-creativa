# Visual Creativa - Sitio Web

Sitio web corporativo de Visual Creativa - Agencia de Comunicación y Producción Audiovisual en Chincha.

## Despliegue en Vercel

Este proyecto está desplegado en Vercel: https://visual-creativa.vercel.app

### Estructura del Proyecto

```
/
├── index.html
├── nosotros.html
├── servicios.html
├── contacto.html
├── css/
│   ├── estilos.css
│   ├── mejoras.css
│   └── ...
├── js/
│   └── script.js
├── img/
├── video/
└── vercel.json
```

### Configuración

El proyecto usa una configuración simple en `vercel.json` para servir archivos estáticos.

## Desarrollo Local

Para ejecutar localmente:

```bash
npm install
npm start
```

O simplemente abre `index.html` en tu navegador.
